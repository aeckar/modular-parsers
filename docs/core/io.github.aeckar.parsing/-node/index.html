Node

open class Node<MatchT : Symbol>

A matched substring in a given input produced according to the matching logic of a symbol.

When nodes are combined into a hierarchy, they form an abstract syntax tree.

Because extension functions of this class are specific to the parameter MatchT, performing a cast to an instance with another type parameter may break the API.
